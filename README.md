# TikTok Clone App

This project is a TikTok-style application built using React Native and Expo.

## 🚀 Features
- Full screen video feed
- Vertical scrolling
- Auto-play videos
- Like, Comment, Share UI

## 🛠️ Tech Stack
- React Native
- Expo

## 📱 Output
A working TikTok-like interface with scrolling video playback.

## ▶️ How to Run
1. Install dependencies:
   npm install

2. Start app:
   npx expo start

3. Open:
   - Browser (press `w`)
   - Expo Go (scan QR)
